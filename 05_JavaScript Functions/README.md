# JavaScript Functions

This folder teaches basic JavaScript functions for beginners.

## What you will learn

- How to write a normal function using `function name(...) { ... }`
- How to pass values into a function with parameters
- How to return a value from a function
- How to use default parameters
- How to write arrow functions
- How to use simple `if` / `else` logic inside a function

## Files in this folder

- `index.js`
  - Shows function declaration, printing values, using default parameter values, and arrow functions.
- `arrow.js`
  - Shows arrow function examples and a small even/odd number checker.

## How to run the examples

Open a terminal in this folder and run:

```bash
node index.js
node arrow.js
```

## Tips for beginners

- A function is a reusable block of code.
- A parameter is a name that the function uses for a value it receives.
- `return` sends a value back from the function.
- Arrow functions are a shorter way to write functions in JavaScript.
- Comments in code help explain what each line does.

## What to try next

- Change the names passed to the `Nice()` function and run `node index.js` again.
- Change the numbers passed to `square()` and `multiply()` in `arrow.js`.
- Try adding a new function to return the difference of two numbers.
